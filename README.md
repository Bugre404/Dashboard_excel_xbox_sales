
#  DIO - Excel Dashboard
um Dashboard de vendas para auxiliar tomadas de decisões

## RESUMO 🖥️

 um dashboard de vendas, com foco na organização e visualização de dados. O objetivo é transformar dados brutos em informações visuais claras e úteis, permitindo uma análise eficaz do desempenho de vendas e a tomada de decisões baseadas em dados.
